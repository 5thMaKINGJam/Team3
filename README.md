# Team3 - '사신도' Game Development Project

## 프로젝트 개요
'사신도'는 한국 신화를 기반으로 한 게임으로, 플레이어가 한국의 전통 문화와 신화를 탐험하는 경험을 제공합니다.

## 팀 구성원 및 역할
- **기획**: 서지희  
  - 게임의 컨셉과 스토리 기획 담당

- **그래픽**: 강수빈  
  - 캐릭터 디자인 및 게임 배경 아트 제작 담당

- **코더**: 황영서, 유지우  
  - 게임 기능 개발 및 코드 작성

- **사운드**: 김류빈  
  - 배경 음악과 효과음 제작
